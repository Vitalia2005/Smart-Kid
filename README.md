Используется: Python, PyQt, SQL, SQLite
Обучающее приложение для детей. Реализовано на Python с помощью библиотеки PyQt, есть база
данных и регистрация пользователей. Прогресс каждого ученика сохраняется. Ребенок может
выучить алфавит, новых животных и цифры. Все кнопки и действия озвучены для детей, не
умеющих читать.
